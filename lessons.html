ICT LESSONS

LESSON 1

HTML (Structure Layer) – This forms the foundation of a webpage. It defines the structure and content, such as text, images, links, and headings. Example: It tells what elements are on the page.
CSS (Style Layer) – This controls the appearance or design of the webpage. It changes how elements look, such as colors, fonts, layouts, and spacing. Example: It makes the page attractive and organized.
JavaScript (Behavior Layer) – This adds interactivity and dynamic features to a webpage. Example: It allows buttons, animations, and forms to respond to user actions.

Style text – change color, font, size, and alignment. Design layouts – use grids, flexbox, and positioning for page structure. Add colors and backgrounds – apply gradients, images, and transparency. Control spacing – adjust margins, padding, and borders. Make websites responsive – adapt layouts for phones, tablets, and desktops I learned that a webpage has three main layers: HTML, CSS, and JavaScript. HTML builds the structure and content, CSS designs and styles the page, and JavaScript adds interactivity and dynamic features. These layers work together to create a complete and functional website.

LESSON 2




I learned that there are three types of CSS: inline, internal, and external. Inline CSS styles a single element, internal CSS styles elements within one page, and external CSS is linked from another file and can style multiple pages.

LESSON 3

I learned that the div tag is used to group large sections of content, while the span tag is for styling small parts of text or inline elements. The display property controls how elements appear on the page, such as block, inline, or none

LESSON 4

learned that CSS classes are used to apply the same style to multiple elements. By using a period (.) before the class name, I can create a rule that changes the design of all elements with that class.

LESSON 5

I learned that CSS selectors are used to choose which HTML elements to style. Examples include element selectors (like p), class selectors (.class), and ID selectors (#id).

LESSON 6

I learned that selecting dimensions in CSS means setting the width and height of elements. This helps control the size, layout, and spacing of objects on a webpage.
